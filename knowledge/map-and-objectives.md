# Deadlock マップ・オブジェクト情報

> 最終更新: 2026-03-02

## マップ名: The Cursed Apple

ニューヨークをモチーフにした都市型マップ。

## レーン構成

マップは **3本のレーン** で構成される。

| レーン | 通称 | 色 |
|--------|------|-----|
| York Avenue | イエローレーン | 黄色 |
| Broadway Street | ブルーレーン | 青 |
| Park Avenue / Greenwich | グリーンレーン | 緑 |

各レーンには対応する色のジップラインが配置されており、素早い移動が可能。

## マップ構成要素

### ジャングルエリア
- 3本のレーンの間に位置するエリア
- ニュートラルのデニズン（敵性生物）キャンプが点在
- 倒すとSoulsを獲得できる

### トンネルと路地
- ゴールデンスタチュー（壊すとSouls獲得）
- 木箱（破壊可能オブジェクト）
- ニュートラルユニット

## 破壊可能オブジェクト詳細

### ガーディアン（Guardian）

各レーンに1体ずつ配置されるレーンの守護者。

| スタッツ | 値 |
|---------|-----|
| 体力 | 5,500 HP |
| ダメージ耐性 | 75% → -50%（12分かけて低下） |
| 銃弾耐性 | 0% |
| DPS | 116 |
| 射程 | 34m |
| 撃破報酬 | 1,000 Souls |

**特性:**
- 敵のトルーパーまたはプレイヤーが24m以内にいない場合は**無敵**になる
- 近距離（5m以内）で近接攻撃を繰り返す
- パリィされると2秒スタン → 5秒間近接攻撃不可

**ベースガーディアン（変種）:**
- 体力: 5,500 HP
- 銃弾耐性: 40%（敵ヒーロー1人ごとに20%低下）
- 撃破報酬: 750 Souls

---

### ウォーカー（Walker）

ガーディアンを破壊した後に露出する強力な機動要塞。

| スタッツ | 値 |
|---------|-----|
| 体力（残り3体時） | 6,800 HP |
| 体力（残り2体時） | 9,350 HP |
| 体力（残り1体時） | 11,900 HP |
| ダメージ耐性 | 75% → -50%（20分かけて低下） |
| 銃弾耐性 | 15% |
| DPS | 125 |
| 射程 | 24m |
| 近接攻撃 | 300ダメージ |
| 撃破報酬 | 3,500 Souls |

**攻撃パターン:**
1. **ファイアビーム**: 最大HP秒あたり2%のチャネルダメージ
2. **ストンプ**: 15m範囲に350ダメージ＋最大HP15%ダメージ、2秒スタン
3. **ファイアボール（ロケットバレッジ）**: 6発弾幕、各200ダメージ、5秒間炎エリア（25ダメージ/秒）

**報酬配分:**
- 40%: 近くのプレイヤー間で分配
- 60%: チーム全体で等分

---

### パトロン（Patron）

最終目標。パトロンを破壊したチームが勝利。

| スタッツ | 値 |
|---------|-----|
| 体力 | 12,000 HP |
| 体力回復 | 120 HP/秒 |
| 攻撃範囲（第1段階） | 76m |

**第1段階（無敵状態）:**
- 2つの聖殿が破壊されるまで無敵
- レーザービーム: 440 + 最大HP5% DPS
- 聖殿防衛: 爆発攻撃

**第2段階（弱体化状態）:**
- HP50%以下で球体化した「弱体化パトロン」に変形
- ダメージパルス: 75ダメージ/秒
- 落とし穴爆発攻撃

## オブジェクト破壊の順番

勝利するための標準的な流れ：

```
トルーパー確保 → ガーディアン破壊 → ウォーカー破壊 → 聖殿破壊 → パトロン撃破
```

## パワーアップバフ（5分ごとにスポーン）

| バフ名 | 効果 |
|-------|------|
| Casting Buff | クールダウン短縮 ＋ スピリットパワー増加 |
| Gun Buff | 射撃速度増加 ＋ 弾薬増加 |
| Movement Buff | スタミナ増加 ＋ 移動速度増加 |
| Survival Buff | 体力増加 ＋ 体力再生 |

## ソウルアーン（Soul Urn）

- **スポーン時刻**: 10分〜（5分ごと再スポーン）
- **着地時間**: 12.5秒後に着地
- 撃破するか回収することで大量の Souls を獲得

## ミッドボス

- **スポーン時刻**: 10:00〜
- 中央エリアに出現する非常に強力なボス
- 大きなHPプール、常時再生シールド、危険なレーザーを持つ
- 撃破したチームに大きなアドバンテージ
