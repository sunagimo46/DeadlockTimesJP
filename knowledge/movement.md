# Deadlock 移動システム完全ガイド

> 出典: https://deadlock.wiki/Portal:Movement, https://deadlock.wiki/Movement, https://deadlock.wiki/Stamina
> 最終更新: 2026-03-02

## 移動システムの概要

Deadlock はSource 2エンジンを採用したシューターであり、移動の自由度が非常に高い。
**落下ダメージが存在しない**ため、高所からの飛び降りやアグレッシブな立体移動を恐れずに行える。

テクニックは大きく4カテゴリに分類される：

| カテゴリ | 内容 |
|---------|------|
| Universal Tech | 全ヒーローで使える汎用テクニック |
| Character Tech | 特定ヒーロー固有の移動アビリティ |
| Item Tech | アイテムによる移動強化 |
| Status Effects | 移動に影響するデバフ・バフ |

## スタミナシステム

移動アクションの燃料となるリソース。ほぼすべての高度な移動はスタミナを消費する。

### 基本仕様

- **回復時間**: 1バーあたり **4.5秒**（一部例外あり）
- **例外**: Bebop・Kelvin は **5.5秒**、Mina は **3.9秒**

## 基本移動アクション

### スプリント
- 戦闘から**5秒後**に自動発動する高速移動
- 攻撃・被弾・エイム時にリセット

### ダッシュ
- 素早い方向転換・回避アクション
- 地上ダッシュ: **固定10m**、速度14.3〜16.1 m/s
- 空中ダッシュ: **固定8m**

### ジャンプ・二段ジャンプ
- 地上ジャンプ: スタミナ消費なし
- 二段ジャンプ（エアジャンプ）: 1バー消費

### マントリング
- 段差・縁を掴んで乗り越える
- 届く高さであれば自動的に発動

### スライド
- クラウチ（Ctrl）を押しながら走ると発動
- 地形の傾斜を滑り降りる際に速度が増加

### クラウチ
- 姿勢を低くして被弾判定を小さくする
- スライド・各種テクニックのトリガーとしても機能

## マップギミックによる移動

### ジップライン
ゲーム中最も重要な移動手段の1つ。レーンカラーに対応した色で設置されている。
- 乗車で高速移動
- 降車時の勢い（モメンタム）を保持するテクニックあり

### テレポーター
- マップ左右のエリアを瞬間移動でつなぐ
- ローテーション（レーン間移動）に活用

### ロープ
- 縦方向の移動を補助
- 建物の上下間移動に使用

### ジャンプパッド
- 乗ることで高く跳躍できる設備

## 中級テクニック

### ダッシュジャンプ（Dash Jump）

**最も基本的な高度移動テクニック。**

- **操作**: ダッシュ → 0.3秒後の0.2秒ウィンドウ内でジャンプ
- **スタミナコスト**: 2バー
- **速度**: 即座に **611 u/s** を適用
- **判定方法**: スタミナバーの四分円が青くフラッシュするタイミングでジャンプ
- **注意**: 前の勢いは上書きされる（モメンタム継続には不向き）

**移動シーケンスの起点として使うのが最適。**

#### 応用: Coyote Dash Jump
崖から飛び出した直後もダッシュジャンプのウィンドウが開いているため、空中でも発動可能。エアジャンプと壁ジャンプを温存できる。

### 壁ジャンプ（Wall Jump）

**スタミナ効率が非常に高い垂直・水平移動テクニック。**

- **操作**: 空中で壁に近い状態でジャンプ入力
- **スタミナコスト**: 初回0バー、以降0.5バー
- **角度調整**: 移動キーの方向で調整可能
  - 壁方向への入力: 垂直成分が高い（高く登る）
  - 壁から離れる方向: 水平速度が高い（遠くへ飛ぶ）

#### Wall Jump Fatigue（疲労メカニクス）
連続使用で威力が低下する。
- 疲労期間: **1.25秒**
- 着地すると疲労状態はリセット

#### 応用テクニック
| テクニック | 説明 |
|-----------|------|
| Corner Boost | 角（コーナー）で壁ジャンプすると通常より高い垂直速度を獲得 |
| Edge Boost | 縁でのジャンプで追加速度を得る |
| Fast Climb | 壁ジャンプを高速で繰り返して素早く登る |
| Zipline Momentum Conservation | ジップラインからの降車時に勢いを保持 |

### スーパーグライド（Superglide）

**スタミナ不要で大きな勢いを得られるテクニック。**

- **操作**: マントリング中 → 動けるようになった瞬間にジャンプ
- **スタミナコスト**: **0バー（無消費）**
- **効果**: マントリングの初期勢いを空中でそのまま維持
- **速度**: ダッシュジャンプに近い距離をスタミナなしで移動可能
- **難易度**: 初級（Universal Tech）

**スタミナが切れているときの緊急回避・距離稼ぎに有効。**

## 上級テクニック

### スライドバニーホップ（Slide Bunnyhop）

**スタミナを消費せず速度を維持し続けられる最効率移動テクニック。**

バニーホップとはQuakeシリーズ由来の、地面に着地するたびにジャンプすることで速度を維持する移動テクニック。

- **操作**: スライド → スライドキー（Ctrl）を押しながらジャンプ → 着地後スライドを繰り返す
- **スタミナコスト**: **0バー（無消費）**
- **必要速度**: 376 u/s 以上
- **注意**: スライド時間が0.5秒未満だと速度が大幅に低下する

**「最もスタミナ効率が良いバニーホップ手法」として評価されている。**

#### 仕組み
スライドアクション後にキャラクターに内部フラグが付与され、スライド中のジャンプが有効になる。空中の摩擦係数が地面より低いため、速度が維持・増加しやすい。

### ダッシュジャンプ→スライドバニーホップ（Combo）

- **操作**: ダッシュジャンプで勢いをつける → 着地時にスライドバニーホップへ移行
- ダッシュジャンプの速度（611 u/s）をスライドバニーホップで維持し続ける

### エアストレーフィング（Air Strafing）

空中で移動方向を滑らかに変える技術。
- ジャンプ中にマウスと移動キーを組み合わせて円弧を描くように動く
- スタミナ消費なしで方向転換が可能

## 戦闘での移動活用

### ピーク（覗き打ち）のコツ
- **右肩カメラ**の仕様により、**右側からのピークが有利**
- 壁の右端から射撃すると自分の被弾範囲が最小になる

### 移動と射撃の両立
- ダッシュ中も射撃可能（一部ヒーロー除く）
- スライド中のエイムで地面に近い低い姿勢から攻撃できる

### 逃げ・追いかけのパターン
- **逃げ**: スーパーグライド → スライドバニーホップでスタミナ消費なしの持続移動
- **追いかけ**: ダッシュジャンプで初速をつけてそのままバニーホップ
