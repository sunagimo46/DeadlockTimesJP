# Deadlock ゲームメカニクス

> 出典: https://deadlock.wiki/Mechanics, https://deadlock.wiki/Trooper
> 最終更新: 2026-03-02

## Souls（魂）システム

ゲームの核となるリソース。アイテム購入・アビリティアップグレードに使用する。

### Souls の獲得方法

| 獲得源 | 詳細 |
|--------|------|
| トルーパーへのラスティングヒット | 確定獲得（最後の一撃） |
| 近接攻撃でトルーパー撃破 | 100%をその場で獲得 |
| 敵ヒーローを倒す | 多量のSoulsを獲得 |
| ゴールデンスタチュー破壊 | 小〜中量のSouls |
| ニュートラルキャンプ | ジャングルのデニズン撃破 |
| ソウルアーン | 大量のSoulsを保有（10分〜） |
| パワーアップバフ | 間接的な強化 |

### トルーパーのSouls報酬

```
1体あたりの報酬 = 116 Souls + (1.16 × 経過分数) Souls
```

**報酬の分配（射撃撃破時）:**
- 60%: 地面のオーブ（拾う必要あり）
- 40%: 浮遊するオーブ（自動獲得）

**近接攻撃で撃破した場合:**
- 100%をその場で直接獲得（効率的）

## トルーパー（Trooper）システム

### 種類

| 種類 | HP | 特徴 |
|------|-----|------|
| 射撃型トルーパー | 標準 | 射程25m、最も一般的 |
| 近接型トルーパー | 400 HP | 近距離特化、移動速度8.1 m/s |
| 医療型トルーパー（バナートルーパー） | 標準 | 撃破時に医療パックをドロップ |

### スポーンパターン

| 時間帯 | 構成 | 間隔 |
|--------|------|------|
| 序盤（〜5分） | 射撃型3体 ＋ 医療型1体 | 30秒/波 |
| 5分以降 | 射撃型2体 ＋ 近接型1体 ＋ 医療型1体 | 30秒/波 |
| 20分以降 | 上記と同様 | 25秒/波（加速） |

## 移動システム

プレイヤーは多彩なアクションで移動できる。すべての行動は **スタミナ** を消費する。

### 移動アクション

| アクション | 説明 |
|-----------|------|
| ダブルジャンプ | 空中でもう一度ジャンプ |
| ダッシュ | 素早い方向転換・回避 |
| エアダッシュ | 空中でのダッシュ |
| ダッシュジャンプ | ダッシュ中に高く跳躍 |
| スライド | 素早いスライディング移動 |
| ウォールジャンプ | 壁を蹴って跳躍 |
| マントル | 段差・縁を掴み乗り越える |
| グライド（一部ヒーロー） | 空中での滑空 |

### ジップライン
- 各レーンに設置された高速移動用設備
- レーンカラーに対応した色
- 移動速度が大幅に向上

## 戦闘システム

### 武器攻撃
- すべてのヒーローが固有の武器（主に銃）を保有
- 基本射撃と近接攻撃の両方を使用可能

### 近接攻撃

| 攻撃種 | 速度 | ダメージ |
|--------|------|---------|
| 軽攻撃（Light Melee） | 速い | 低い |
| 重攻撃（Heavy Melee） | 遅い | 高い |

### パリィ（Parry）
近接攻撃をタイミングよくブロックする防御テクニック。

**効果:**
- 攻撃者をスタン
- スタン中の攻撃者に **被ダメージ25%増加（2.75秒間）**
- パリィが成功したガーディアンは2秒スタン → 5秒間近接不可

## アビリティシステム

### 基本構成
- 各ヒーローは **4つのアビリティ** を保有
- ゲーム開始時から特定のアビリティが使用可能
- Souls を消費してアップグレード

### アップグレード
- 各アビリティに複数のアップグレードティアが存在
- ダメージ増加・効果範囲拡大・クールダウン短縮などが強化される

## カムバックメカニクス（弱者救済）

敗色濃厚なチームへの支援システム。

| 条件 | 内容 |
|------|------|
| チームが大きくリードされている | トルーパーからのSouls最大30%ボーナス |
| 敵ヒーローを倒した時 | より多くのSoulsを獲得 |
| ソウルアーン配置 | チームのネットワース差に応じて配置が変化 |

## マップ内のインタラクティブ要素

| 要素 | 効果 |
|------|------|
| ソウルアーン | 大量のSouls（10分〜） |
| 破壊可能オブジェクト | 小量のSoulsやボーナス |
| Sinner's Sacrifice | 体力をSoulsに変換 |
| ジップライン | 高速移動 |
| テレポーター | 瞬間移動 |
| ロープ | 縦方向の移動補助 |

## ショップシステム

- マップ内の各レーンにショップが存在
- ゲーム中いつでもアイテム購入可能
- 購入はリスポーン中や移動中でも可能（ゲーム内UI経由）

## 戦略の基本概念

### レーニング
- 指定されたレーンに配置されて序盤を戦うフェーズ
- トルーパーのSoulsを効率よく獲得することが最重要

### ガンキング（Ganking）
- 担当レーンを離れ、他のレーンへ援護に行く行動
- 敵を奇襲してキル・アドバンテージを作る

### フリーファーム
- 敵がいないエリアでトルーパーを安全に倒してSoulsを稼ぐ

### オブジェクトコントロール
- ミッドボス・パワーアップ・ガーディアンの確保が勝利への近道
