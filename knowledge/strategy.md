# Deadlock 基本戦略

> 出典: https://deadlock.wiki/Strategy
> 最終更新: 2026-03-02

## 基本目標

1. 3本のレーンでトルーパーを率いて戦線を押し上げる
2. 敵のガーディアン・ウォーカーを破壊してベースへ侵攻
3. 聖殿を破壊してパトロンへのアクセスを確保
4. 敵のパトロンを撃破して勝利

---

## レーン編成

**標準編成: 2-2-2**（各レーンに2人）

- **York Avenue**（イエロー）
- **Broadway Street**（ブルー）
- **Park Avenue / Greenwich**（グリーン）

### ペアの組み方
- サポート型ヒーロー ＋ 高DPSヒーローの組み合わせが最適
- 例: DynamoのSingularityで敵を集約 → Infernusが爆発させるコンボ

---

## ゲームフェーズ

### 序盤（レーニング）

**優先事項:**
1. トルーパーへのラスティングヒットでSoulsを確実に獲得
2. 敵ヒーローへの圧力をかけつつ生存
3. 小型デニズンキャンプを効率よく刈る（フリーソウル）

**ジャングルキャンプのSouls収益（予定スケジュール）:**

| タイミング | 獲得Souls |
|-----------|---------|
| 2分 | 177 Souls |
| 4分 | 180 Souls |
| 6分 | 186 Souls |
| 8分 | 189 Souls |
| 合計（10分まで） | 732 Souls |

### ミッドゲーム

**優先事項:**
1. ガーディアン破壊でExtra Slotsを獲得
2. ミッドボス（10:00〜）の確保
3. ソウルアーン（10:00〜）の回収
4. 他レーンへのローテーション（ガンク）

### 終盤

**優先事項:**
1. ウォーカー破壊
2. 聖殿（Shrine）の破壊
3. パトロン撃破

---

## チーム構成の考え方

相互補完的なヒーローの組み合わせが重要。

**コンボ例:**
- **Dynamo ＋ Infernus**: SingularityでまとめてからInfernusの爆発でまとめて撃破
- 「CC（行動不能）＋ 高火力」の組み合わせが基本

**役割のバランス:**
- タンク・フロントライン: 1〜2人
- ダメージディーラー: 2〜3人
- サポート・イニシエーター: 1〜2人

## ミクロ戦術

### Uptime（稼働率）
敵への圧力をかけながらSoulsを収集する時間を最大化する。
戦闘・ファーム・ローテーションの無駄を減らすことが上達への近道。

### Positioning（ポジショニング）
マップ制圧と戦闘効率を高める立ち位置の選択。

### カバーの活用
**重要テクニック**: 右肩カメラの仕様により、**右側からのピーク（覗き打ち）が有利**。
壁の右端から射撃すると、自分の体を晒す範囲が最小になる。

### トルーパーマネジメント

| 戦術 | 説明 |
|------|------|
| **デナイ（Denial）** | 味方トルーパーを撃ち込んで、敵がSoulsを獲得するのを妨害する |
| **フリーズレーン（Freezing Lane）** | 敵・味方のトルーパー数を均衡させて、自陣寄りでウェーブを止める |
| **ウェーブクリア** | 素早くトルーパーを一掃して次の行動に移る |

### Box Running（ボックスランニング）
マップ内の破壊可能オブジェクト（木箱・ゴールデンスタチュー）を周回して素早く回収するテクニック。

### Jungling（ジャングリング）
デニズンキャンプを継続的にクリアして、レーン外でもSoulsを稼ぐ行動。

### Funneling（ファネリング）
1人のキャリーヒーローにSoulsを集中させる戦略。サポートが自分のSoulsを譲渡し、キャリーを最速で育てる。

### Rotations（ローテーション）
自分のレーンから他のレーンへ移動してガンクを行う。
タイミング: ウェーブをクリアした直後・レーンが安定しているとき。

## カウンターアイテム指針

敵のビルドや戦略に応じて購入するアイテム。

| 対策したい状況 | 推奨アイテム |
|--------------|------------|
| DoT（継続ダメージ） | Debuff Reducer、Dispel Magic |
| 高バースト火力 | Metal Skin、Battle Vest |
| 回復・ライフスティール | Healbane、Decay |
| 機動力の高い敵 | Slowing Hex |
| アビリティ多用の敵 | Silence Wave |

---

## 重要な戦略原則

1. **レーンの押し上げ方**: 敵トルーパーとオブジェクトを倒すことでレーンが進む
2. **スノーボール**: 早期のSoul優位が加速度的な強化につながる
3. **アルティメット獲得タイミング**: アルティメットを最初に取ることがスノーボールの鍵

## 用語集

| 用語 | 意味 |
|------|------|
| **デナイ（Denial）** | 敵がSoulsを取れないよう、味方トルーパーをわざと撃って倒す行動 |
| **ガンク（Gank）** | 他レーンへ移動して敵を奇襲すること |
| **ローテーション** | レーン間の移動 |
| **フリーズ** | ウェーブをレーン前半に留める行動 |
| **ジャングリング** | ジャングルのデニズンキャンプを狩ってSoulsを稼ぐ |
| **ファネリング** | 1人のキャリーにSoulsを集中させる戦略 |
| **Box Running** | 破壊可能オブジェクトを周回回収する行動 |
| **Uptime** | プレイの生産性・稼働率 |
| **スノーボール** | 序盤の優位が加速度的に拡大していく現象 |
| **キャリー** | 後半に強くなる火力型ヒーロー |
