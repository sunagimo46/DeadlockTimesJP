## 主要情報サマリー

- **ゲーム形式**: 6vs6 三人称視点ヒーローシューター（MOBA要素あり）
- **開発元**: Valve
- **勝利条件**: 敵のパトロン（Patron）を破壊
- **ヒーロー数**: 38体
- **アイテム数**: 172種類（Weapon / Vitality / Spirit の3カテゴリ）
- **レーン数**: 3本（York Avenue / Broadway / Park Avenue）
