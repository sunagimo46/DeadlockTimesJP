## Deadlockとは

**Deadlock**はValveが開発した6対6のヒーローシューターです。
MOBAとサードパーソンシューターを組み合わせた独特のゲームスタイルが特徴で、**銃を撃ちながらアビリティを使い、ソウルを稼いでアイテムを購入していく**ゲームです。

**最終目標は敵チームのパトロンを破壊すること**です。

---

## ソウルとは

ソウルはこのゲームにおける**通貨兼経験値**です。ソウルを稼いでアイテムを買い、ヒーローを強化することがゲームの根幹です。

主な獲得源は以下の通りです：

| 獲得源                           | 種類   | 備考                                                   |
| -------------------------------- | ------ | ------------------------------------------------------ |
| トルーパー（ミニオン）の撃破     | 確定   | トルーパーを倒すとオーブが出現し、取得するとソウル獲得 |
| 敵ヒーローの撃破                 | 確定   | まとまったソウルを獲得                                 |
| 木箱・ゴールデンスタチューの破壊 | 未確定 | スキマ時間に積極的に回収                               |
| ジャングルキャンプ               | 未確定 | マップ各所に出現するニュートラルユニット               |

### 未確定ソウル（Unsecured ソウル）

ジャングルキャンプや木箱などから獲得したソウルは**未確定状態**になります。

- ショップでのアイテム購入には使えますが、**確定ソウルから先に消費**されます
- 時間経過とともに自動的に確定オーブになっていきます
- **未確定ソウルを持った状態で死亡すると、その場にオーブとしてドロップ**されます（敵に拾われる可能性があります）

<Callout type="warning">

未確定ソウルが多い状態で前線に出るのは危険です。死亡リスクが高い場面の前にアイテムを購入して消費しておきましょう。

</Callout>

### 浮遊オーブとグラウンドオーブ

トルーパーを倒すと2種類のオーブが出現します。

**浮遊オーブ（Floating Orbs）**

空中に浮かぶオーブです。最初は黒色で、しばらくすると光って**射撃で取得可能**になります。

- 味方のオーブは**緑色**、敵のオーブは**橙色**で表示されます
- 敵の浮遊オーブを撃つと**deny**できます（敵のソウルを奪える！）
- denyは相手の成長を妨害する重要テクニックです

**グラウンドオーブ（Ground Orbs）**

地面に落ちるオーブです。トルーパー討伐時のソウルのうち約60%がこちらとして分配されます。

- denyはできず、**近づくだけで自動取得**されます
- 通常は18秒間、試合10分経過後は最大40秒間地面に残ります

<Callout type="tip">

レーン戦では自分のオーブを取るのになれたら
敵の浮遊オーブを射撃してdenyすることも意識しましょう。差がつく重要ポイントです。

</Callout>

### ソール分配

同じオーブを複数の味方が近くにいる状況で取得すると、**ソウルが分配**されます。

| 関与人数 | 1人あたりの取得割合 |
| -------- | ------------------- |
| 1人      | 100%                |
| 2人      | 54%                 |
| 3人      | 36%                 |
| 4人      | 25%                 |

序盤のレーン戦では1対1を維持することで、効率よくソウルを稼げます。

## ヒーローの選び方

難しく考える必要はありません。**見た目や雰囲気が好きなヒーローを選んでください。**

強いて言えば、初心者に扱いやすいヒーローと扱いにくいヒーローがあります。
個人的なおすすめのヒーローです。参考にしてください。

| ヒーロー | 備考                                                                  |
| -------- | --------------------------------------------------------------------- |
| Seven    | 火力が高めで、ジャングルでの成長ができ、ULTが爽快。きもちいい。       |
| Mo&Krill | 耐久力が高く回復もあり、ULTがわかりやすい。火力は低め。               |
| Warden   | Moほどではないが耐久力があり、単体での火力がある。ULTは少し癖がある。 |
| Infernus | 火力があり、移動スキルが強くやべってなってから逃げやすい。            |

### おすすめしにくいヒーロー

| ヒーロー | 備考                                                                                              |
| -------- | ------------------------------------------------------------------------------------------------- |
| Kelvin   | スキルは強いが、ULTで味方チームを崩壊させてしまう場合があり、ゲームを理解するまでおすすめしない。 |
| McGinnis | 一見初心者向きに見えるが、機動力が少なく敵から逃げるのが難しい。                                  |
| Mina     | スキルが強くうまく立ち回ればとても強いが、耐久力がなく、油断するとすぐデスしてしまう。            |
| Sinclair | スキルが非常に癖が強く、ULTは敵のULTをコピーするためヒーロー理解が必要                            |

ただし、**好きなヒーローを使い続けることが一番の上達方法**です。
スキルが複雑でも、好きなヒーローであれば自然と覚えていけます。

## 移動の基本

Deadlockは**移動システムが非常に優れたゲーム**です。最初に押さえておくべき要素を紹介します。

### スプリント

戦闘から5秒離れると自動で高速移動（スプリント）に切り替わります。攻撃・被弾・エイム時にリセットされます。

### スタミナ

移動アクションの燃料となるリソースです。

| アクション                   | スタミナコスト  |
| ---------------------------- | --------------- |
| 通常ダッシュ                 | 1バー           |
| ダッシュジャンプ             | 2バー           |
| エアダッシュ（空中ダッシュ） | 1バー           |
| ダウンダッシュ               | 0.5バー         |
| 壁ジャンプ（初回）           | 0バー（無消費） |

**初心者のうちはまず「壁ジャンプ」を覚えましょう。** 初回は０コストで使用でき、マップ移動の基本テクニックになります。

<Callout type="tip">

スタミナは資源です。無闇に使い続けず、逃げや追いかけに備えて温存する意識を持ちましょう。
ただ慣れるまでは使いまくったほうが覚えやすいです。

</Callout>

### ジップライン

基本的なマップ移動手段です。

- 降りるときは**しゃがみキー（デフォルト: Ctrl）で降りる**と慣性を保てより遠くへ移動できます。
- 乗ろうとしたときに攻撃を食らうと**スタンして大ダメージを受ける**ので注意

### テレポーター・ジャンプパッド

- **テレポーター**：マップの左右を瞬間移動できます。慣れてないと使いこなせないの移動手段なので、場所を覚えておくだけで移動が格段に楽になります
- **ジャンプパッド**：マップ各所に設置されており、乗ると高く跳び上がれます。移動ルートとして積極的に使いましょう。慣れるとジップラインより早いです。

## アイテム購入

最初のうちは**ゲーム内のポピュラービルドをそのまま使う**のがおすすめです。ビルドセクションを開き、人気ビルドを選んで左上から順に購入するだけでOKです。

### アイテムスロット

ゲーム開始時は**9つのアイテムスロット**があります。ウォーカーを1体倒すごとに追加スロットが1つ解放され、最終的に12スロットになります。
