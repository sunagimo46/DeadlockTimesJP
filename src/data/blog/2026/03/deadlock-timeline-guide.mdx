## ゲーム開始〜10分：レーニングフェーズ

序盤の最大の目標は**ソウルを稼ぐこと**です。

### ウェーブを押し込む

Deadlockでは一般的なMOBAとは異なり、**ウェーブを積極的に敵タワーへ押し込むことが有利**につながります。
トルーパーをできるだけ速く処理して前に進みましょう。

### デナイ（妨害）を意識する

自分がソウルを取るだけでなく、**敵がソウルを取るのを妨害**することも重要です。
敵の浮遊オーブに先に撃ってソウルを奪いましょう。

### 箱とゴールデンスタチューを回収する

**木箱**を壊すとソウルが手に入ります。
近くにある**金色のゴールデンスタチュー**を壊すと永続バフを獲得できます。
トルーパーのウェーブとウェーブの間に取ると敵と差が付きます。

### 積極的にプレイする

怖がって消極的になると上達が大幅に遅れます。ミスをしながら覚える方が圧倒的に早く上達できます。まずはどんどん動いて試してみましょう。

## ガーディアン攻略

各レーンに存在する防衛タワーです。**ある程度近づかないと攻撃が通りません**。レーンを押し込んでガーディアンに接近した状態で攻撃しましょう。

### 近接攻撃はパリィで反撃できる

ガーディアンに近づくと**近接攻撃**を繰り返し使ってきます。この攻撃は**パリィで弾くことができます**。

パリィに成功すると：

- ガーディアンが**2秒間スタン**する
- その後**5秒間は近接攻撃ができなくなり**、レーザー攻撃に切り替わる

### 急いで攻略する必要はない

ガーディアンはゲーム開始直後は**75%のダメージ耐性**を持っており非常に硬いですが、**最初の12分間で徐々に耐性が下がっていきます**。

| 経過時間 | ダメージ耐性         |
| -------- | -------------------- |
| 0分      | 75%                  |
| 6分      | 約12%                |
| 12分以降 | -50%（増加ダメージ） |

## 2分：デニズンキャンプ（小）・箱とゴールデンスタチュー出現

**同じレーンに小型キャンプが出現したら、できるだけ早く取りましょう。**
先に取れれば敵より多くのソウルを稼げます。

---

## 5分：パワーアップバフ出現

マップの橋エリアに4種類のバフが出現します。**ヘビーメレー（重攻撃）で取得**します。
ロープやジャンプパッドを使って向かいましょう。

| バフ名        | 効果           |
| ------------- | -------------- |
| Movement Buff | 移動強化       |
| Survival Buff | 体力強化       |
| Casting Buff  | スピリット強化 |
| Gun Buff      | 銃撃強化       |

バフは獲得後**160秒持続**し、その後**5分ごとに再出現**します（10分、15分、20分…）。

---

## 6分：デニズンキャンプ（中）出現

Tier2の中型ジャングルキャンプが出現します。
ほとんどのヒーローで比較的効率よく倒せます。レーンの余裕を見て取りに行きましょう。

---

## 8分：スロットマシン(Sinner's Sacrifice)出現

ヘビーメレーで叩くと**大量のソウルと永続バフ**を獲得できます。

### ジャックポットの取り方

上部のライトが点灯していき、最後の1発でタイミングを合わせると**ジャックポット（永続バフ×4）**を獲得できます。

- 残りライトが**3つになったタイミング**でヘビーメレーをチャージし始める
- うまく決まると4つの永続バフを一気に入手

できれば**敵のスロットマシンを奪いに行く**のが理想ですが、難しければ自分のものを確実に取るだけでも十分な価値があります。

---

## 10分：ミッドゲーム開始

10分を境にゲームが大きく変わります。2つの重要オブジェクトが同時に出現します。

### ミッドボス（レジュベネーター）

マップ中央に強力なボスが出現します。倒すと**レジュベネーターバフ**がドロップします。

- ヘビーメレーで叩くと取得、3つまで取得可能
- バフ1つにつき**チームメンバーが死亡したとき1回即座に復活**できる

10分ちょうどに倒しに行く必要はありません。**15〜20分以降、大きなチームファイトに勝った後**に狙うのが基本です。

### ソウルアーン（Soul Urn）

アーンを拾って指定地点まで運ぶと**大量のソウルを獲得**できます。

- 運搬中は武器・アビリティ・アイテムが使えない（スタミナと近接攻撃は使用可）
- ジップラインとテレポーターも使用不可
- **試合の優劣によってゴール場所が変わります**：自チームが優勢→敵陣、拮抗→中央、劣勢→自陣

<Callout type="important">

アーンを無視するプレイヤーが多いですが、優勢でも劣勢でも**アーンを取ることは非常に重要**です。敵が数人死亡した隙やウェーブをすべて処理した後に積極的に狙いましょう。

</Callout>

---

## ミッドゲーム（10分〜）の動き方

ミッドゲームは混乱しやすいですが、以下の2点だけ意識すれば十分です。

1. **ウェーブを押し上げる**：マップ全体でウェーブが処理されていれば行動の自由度が大きく増します。敵陣までウェーブが押し込めていれば敵のジャングルキャンプを奪うこともできます。
2. **有利な状況で戦う**：ファイトを挑む前に優位性を確認しましょう。ソウル差・人数差（2対1など）・地形優位などを評価して積極的に戦いましょう。

また、同じレーンに留まり続けず、**マップ全体を動き回る**ことを意識してください。
ウェーブを押したらすぐ別のレーンや目標に向かう習慣をつけましょう。

## ウォーカー攻略

ガーディアンを破壊すると次に現れる強力な機動要塞です。倒すと**アイテムスロットが1つ解放**され、大量のソウルも手に入ります。

### 2種類の攻撃を覚える

ウォーカーの周囲には内側と外側、2つの円があります。

| 攻撃                 | 発動条件                 | 対処法                 |
| -------------------- | ------------------------ | ---------------------- |
| ストンプ（踏みつけ） | **内側の円**に入ると発動 | 内側に入らない         |
| ファイアボール       | **外側の円**にいると発動 | 動いて炎エリアを避ける |

ストンプは大ダメージ＋スタンを受けるため特に危険です。

<Callout type="warning">

**13〜14分前は無理にウォーカーを攻めない**ことをおすすめします。序盤は非常に硬く、リスクに見合いません。1人が死んでも倒せるなら価値がありますが、2〜3人を失うようなら待つべきです。

</Callout>

## 終盤〜ゲームの終わらせ方

```
ミッドボス確保 → 聖殿（Shrine）×2 破壊 → パトロン撃破
```

### 聖殿（Shrine）の攻撃を避ける

聖殿は「円が収束するアニメーション」の後に爆発攻撃をしてきます。
**カース（アビリティ・アイテム封印）**と大ダメージを受けるため、円が縮まってきたらすぐ距離を取りましょう。

### パトロンの2フェーズ

| フェーズ | 条件      | 注意点                                       |
| -------- | --------- | -------------------------------------------- |
| 第1段階  | HP50%以上 | リチュアルサークルの外からでも攻撃可         |
| 第2段階  | HP50%以下 | **サークル内に入らないとダメージが通らない** |

パトロンの第1段階を破壊し第二段階にすると敵のリスポーンが最大20秒短縮されるので、敵のリスポーンに間に合わない場合は撤退することも多いです。

第2段階でも聖殿と同様の爆発攻撃があります。円が広がってきたら一度外に出て、爆発後にすぐ戻りましょう。

<Callout type="warning">

**敵が2人以上サークル内にいる場合、パトロンは無敵**になります。まず敵を全員処理してから攻撃しましょう。

</Callout>
